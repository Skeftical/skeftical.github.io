Projects :

  • EDA Analysis for Chicago Crimes
    An EDA analysis for Chicago Crimes focusing on the spatial-temporal dimensions of the dataset. In this kaggle kernel I construct interesting visualisations that answer questions that I had when I first encountered the dataset. I also comment on any on the findings and answer subsequent questions generated.
  • Research Paper Recommender
    This project is aimed at helping researchers and people who read scientific papers find new papers that might be of interest to them. It leverages arXiv and recent developments in word embeddings (using a popular library gensim) to automatically fetch new papers for a given category and then add them to a specified folder if they are found to be similar to a list of given papers already in the user's library.
  • Growing Networks
    An implementation of the algorithm listed in the paper "A self-organising network that grows when required". This algorithm can be used for clustering tasks, vector quantization tasks and dimensionality reduction tasks
  • Cyprus Budget Explorer
     This project aims to build a system that would allow for exploring the budget of the Republic of Cyprus. The data is available in a pdf form from here : http://bit.ly/2biYQ0y Because the data is not available in any other form the pdf file was parsed and cleaned using Tabula and Google Refine. The extracted data is available in a clean format in "proupologismos-revised-csv.csv".
  • MapReduce/HBase code examples
     As part of the Big Data Course, implemented a number of algorithms performing top-k queries, exploratory analysis and more. The code examples, show how to use secondary sorting how to efficiently leverage the clean-up phase of Mappers and Reducers.
